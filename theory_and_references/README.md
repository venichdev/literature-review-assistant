# Theory and References Directory

This directory contains your existing knowledge base - mathematical models, formulas, theories, and references that form the foundation of your research.

## Purpose

The AI assistant will:
1. **Read** existing files in this directory before starting literature reviews
2. **Extract** key concepts, formulas, and references
3. **Use** this knowledge to find related papers and expand your research
4. **Cross-reference** new findings with existing knowledge

## Suggested Files

### 1. `mathematic_model.md`
Contains mathematical formulas, models, and equations relevant to your research.

**Example structure**:
```markdown
# Mathematical Models

## Model Predictive Control Formulation

### Cost Function
J = sum(...)

### Constraints
...

## References
- [Author, Year] - where this formula comes from
```

### 2. `references.md`
Database of important references and citations.

**Example structure**:
```markdown
# Reference Database

## Seminal Works
- [Smith, 2010] - Foundational paper on...
- [Jones, 2015] - Introduced...

## Recent Important Papers
- [Garcia, 2023] - Latest developments in...
```

### 3. `concepts.md` (Optional)
Key concepts and terminology.

### 4. `notes.md` (Optional)
Research notes and observations.

## Usage

When you ask the AI assistant to:
- "Expand on existing research"
- "Find papers related to my notes"
- "Search for applications of [formula from my notes]"

The assistant will first read files in this directory to understand your existing knowledge base.

## File Format

- **Markdown** (`.md`) is recommended for easy reading
- You can also use plain text (`.txt`)
- Include **citations** where applicable
- Use **clear headers** to organize content

## Example

See `mathematic_model.example.md` and `references.example.md` for templates.

---

**Note**: This directory is for YOUR existing knowledge. It's optional - if you're starting a review from scratch, you can leave it empty.
