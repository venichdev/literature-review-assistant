## [1.0.0] - 2025-10-06

- Comprehensive README.md with usage instructions
- INSTRUCTIONS.md for AI assistant guidance
- parameters_topics.md template for defining search criteria
- config.example.yaml with detailed configuration options
- Template files for outputs:
  - paper_note_template.md
  - summary_template.md
  - table_template.md
- Example files for knowledge base:
  - mathematic_model.example.md
  - references.example.md
- .gitignore file for common exclusions
- CONTRIBUTING.md with contribution guidelines
- LICENSE file (MIT)
- theory_and_references/ directory structure

### Documentation
- Detailed workflow process
- Multiple output format examples
- Citation management guidelines
- Best practices for systematic reviews
- Integration with existing knowledge bases

## How to Update This File

When making changes:

1. Add new changes under `[Unreleased]` section
2. Categorize changes as: Added, Changed, Deprecated, Removed, Fixed, Security
3. When releasing, move unreleased changes to a new version section
4. Follow the date format: YYYY-MM-DD

---

**Legend**:
- `Added`: New features
- `Changed`: Changes in existing functionality
- `Deprecated`: Soon-to-be removed features
- `Removed`: Removed features
- `Fixed`: Bug fixes
- `Security`: Security fixes
